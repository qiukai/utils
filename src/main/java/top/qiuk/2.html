54、photoshop图像处理往往会用到通道。通道是什么？有何类型和作用？
参考答案：
       通道是图像某一方面的图像数据。通道有3种类型：彩色通道，alpha通道，专色通道。
        彩色通道是颜色分量图，例如RGB图像有红、绿、蓝三个通道，分别存储所有像素点的红色、绿色、蓝色分量图像，经常在目标和背景差别最大的通道上进行抠图。
        alpha通道，它是黑白图，黑色表示对应的像素点没有选中，白色部分表示选区。alpha通道可用来存储选区，也可利用选区抠图。蒙版的本质就是通道。
        最后是专色通道。它主要用于专色印刷的附加印版，可以保存专色信息，但是每个专色通道只能存储一种专色信息，并且是以灰度形式进行存储的。

57、多媒体数据中的冗余有哪几种？请举例说明。
参考答案：
多媒体数据中的冗余有空间冗余、时间冗余、信息熵冗余、结构冗余、知识冗余、视觉冗余、其它冗余。空间冗余比如图像中空间域或频率域中的冗余数据；时间冗余比如视频相邻帧的相关性，动画和音频也有时间冗余；信息熵冗余，也称编码冗余，指表示数据所用的比特数大于数据的信息熵，比如Huffman编码和算数编码可以消除信息熵冗余；结构冗余指草席这类图像的纹理结构；知识冗余是指某些数据可以由基础知识推导出来，不必保存；视觉冗余是由于人类视觉系统对图像的某些变换不敏感，这类信息丢失人类视觉也觉得图像质量足够好，听觉冗余也类似。

59、简述MPEG-1、MPEG-2、MPEG-4、MPEG-7、MPEG-21各自的用途。
参考答案：
MPEG-1用于大约1.5Mbps速率的数字存储媒体的运动图象及伴音编码，力求把video存储在CD-ROM光盘上，即Video-CD；MPEG-2是高速率编码标准，用于高质量画面和伴音的DVD和HDTV；MPEG-4是窄带宽的视听编码，用于网络视听通信；MPEG-7不是视频编码，而是多媒体内容描述接口，描述媒体特征以方便基于内容的检索；MPEG-21也不是视频编码，而是多媒体框架，试图协调各种多媒体技术标准，建立一个标准化的多媒体框架，以支持各种应用。
60、触摸屏有哪些类型？它们的原理是什么？
参考答案：
        触摸屏按工作原理的不同可分为红外线触摸屏、电容式触摸屏、电阻式触摸屏、表面声波触摸屏和底座式矢量压力测力触摸屏。

        红外线触摸屏，其传感器四周布满红外线发射器和接收器，形成红外线矩阵。将传感器固定在普通显示器的前面，当手触摸时，红外线被遮挡，检测X方向和Y方向被遮挡的位置，就可得到被触摸位置的坐标值。

        电容式。由于人体电场，用户手指和工作面形成一个耦合电容，手指吸收走一个很小的电流，这个电流分别从屏的四个角上的电极中流出，且理论上流经四个电极的电流与手指头到四角的距离成比例，通过对四个电流比例的精密计算，得出位置。

        电阻式。玻璃上下涂覆两层ITO透明导电层，导电层之间留有缝隙互相绝缘。当手指按下，将检测到的电阻值转换成坐标数字信息。

        表面声波触摸屏。将阻挡声波的位置转换成坐标数字信息。

        矢量压力触摸屏将触摸压力点的位置和压力转换成坐标信息。
